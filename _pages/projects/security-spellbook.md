### Download & Install

```bash 
    git clone https://github.com/GouveaHeitor/security-spellbook
    cd security-spellbook

    # Building image
    $ docker build --rm --squash -t kali .

    # Using this container
    $ docker run -p 1337:1337 -p 9090:9090 -ti kali /bin/bash
```
